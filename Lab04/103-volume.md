1. Mongo Volume


    ```bash
    docker run -d mongo:latest
    docker exec -it <<id>> /bin/sh
    mongo
    show dbs
    use shop

    db.products.insertOne({name:"A book B",pice: 11,category:"a"})

    db.products.find().pretty()
    ```
